# ZGU-lab-work-git
# Простой калькулятор на Python

Краткое описание проекта: Это учебный проект — простой консольный калькулятор для демонстрации работы с Git и GitHub.

## Установка и запуск

1. Клонируйте репозиторий
git clone 

2. Перейдите в папку:
cd ZGU-lab-work-git/src

3. Запустите скрипт:
python main.py


## Основные функции

- Вывод приветствия: `print("Hello, World!")`
- Сложение двух чисел: функция `add(a, b)`
- Умножение двух чисел: функция `multiply(a, b)`

### Пример вывода
Hello, World!
2 + 3 = 5
